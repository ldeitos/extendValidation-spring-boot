ExtendedValidation

A BeanValidation Extension

Iniciar

Características

Integrado ao CDI

- Implementação de ConstraintViolationFactory integrado à API CDI.

Parametrização de mensagens

- Suporte estendido a parametrização do texto das mensagens.

Múltiplas fontes de mensagens

- Suporte a parametrização de múltiplos arquivos de mensagens.
- API extensível que permite a substituição da origem padrão das mensagem de validação.

Abstração para validadores

- Simplifica o registro de múltiplas violações em um único validador.
- Permite resolver parâmetros das mensagens de violação em tempo de execução.
- Suporte facilitado ao registro do path da violação.

Validação de parâmetros de métodos através de interceptor CDI

- Disponibiliza um interceptor que permite validar parâmetros de métodos.
- Mecanismo para identificar parâmetros que não devem ser validados.
- Possibilita customizar o tratamento das mensagens geradas durante a validação.

Pure based.
